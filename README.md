# 🎨 Portfólio Pessoal — Henriqcode

## ✨ Sobre o Projeto

Este projeto é meu portfólio pessoal desenvolvido para apresentar minha identidade profissional, habilidades criativas e projetos em desenvolvimento web, design visual e construção de presença digital.

A proposta foi criar uma experiência sofisticada, moderna e visualmente marcante, utilizando uma interface premium com foco em:

* Design minimalista e elegante
* Navegação fluida
* Responsividade
* Performance otimizada
* Identidade visual única
* Estrutura profissional para deploy

---

## Tecnologias Utilizadas

### Frontend:

* HTML5

### Estrutura:

* Organização modular de arquivos
* GitHub Pages para hospedagem
